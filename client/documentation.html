MCU Movies API - Documentation

/getMoviesGETHEAD

Query Parameters

idtype: number (optional) - A movie's ID in the API to filter by
titletype: string (optional) - A movie's title to filter by
actortype: string (optional) - An actor's name to filter by
directortype: string (optional) - A director's name to filter by
yeartype: string (optional) - A release year to filer by

Returns

A JSON array of MCU movie objects matching the paramters
Format: [{ id: number, title: string, phase: string, genres: [string], release date: string, directors: [array], screenwriters: [string], producers: [string], cast: [string], plot: string, languages: [string], filming location: string, status: string, movie rating: string, review rating: number, movie run time: string, budget: number, box office: number }]

Example

Request: /getMovies?title=guardians&year=2014
Result: [{"Id":10,"Title":"Guardians of the Galaxy","Phase":"Phase Two","Genres":["Action","Adventure","Comedy","Sci-Fi"],"Release Date":"01-08-2014","Director":["James Gunn"],"ScreenWriter":["Edgar Wright","Joe Cornish","Adam McKay","Paul Rudd"],"Producer":["Kevin Feige"],"Cast":["Chris Pratt","Vin Diesel","Bradley Cooper","Zoe Saldana"],"Plot":"A group of intergalactic criminals must pull together to stop a fanatical warrior with plans to purge the universe.","Language":["English"],"Filming Locations":"Shepperton Studios, Shepperton, Surrey, England, UK","Status":"Released","Movie Rating":"PG-13","Review Rating":8.1,"Movie Run Time":"121 min","Budget":170000000,"Box office":774176600}]

/getActorGETHEAD

Query Parameters

actortype: string (required) - An actor's name to filter by

Returns

A JSON array of MCU movie objects the actor has been in
Format: [{ title: string, cast: [string], release date: string, movie rating: string, review rating: number }]

Example

Request: /getActor?actor=Vin%20Diesel
Result: [{"Title":"Guardians of the Galaxy","Cast":["Chris Pratt","Vin Diesel","Bradley Cooper","Zoe Saldana"],"Release Date":"01-08-2014","Movie Rating":"PG-13","Review Rating":8.1},{"Title":"Guardians of the Galaxy Vol. 2","Cast":["Chris Pratt","Zoe Saldana","Dave Bautista","Vin Diesel"],"Release Date":"05-05-2017","Movie Rating":"PG-13","Review Rating":7.7}]

/getMoviesByReviewRatingGETHEAD

Query Parameters

greaterOrLesstype: string (required) - Whether you're searching for movies that have a higher (>) or lower (<) rating
ratingtype: number (required) - A rating to filter by

Returns

A JSON array of MCU movie objects matching the paramters
Format: [{ title: string, review rating: number, movie run time: string, release date: string, movie rating: string }]

Example

Request: /getMoviesByReviewRating?greaterOrLess=<&rating=7
Result: [{"Title":"The Incredible Hulk","Review Rating":6.7,"Movie Run Time":"112 min","Release Date":"13-06-2008","Movie Rating":"PG-13"},{"Title":"Captain America: The First Avenger","Review Rating":6.9,"Movie Run Time":"124 min","Release Date":"22-07-2011","Movie Rating":"PG-13"},{"Title":"Thor: The Dark World","Review Rating":6.9,"Movie Run Time":"112 min","Release Date":"08-11-2013","Movie Rating":"PG-13"}]

/getMoviesByRuntimeGETHEAD

Query Parameters

greaterOrLesstype: string (required) - Whether you're searching for movies that have a longer (>) or shorter (<) runtime
runtimetype: number (required) - A runtime to filter by

Returns

A JSON array of MCU movie objects matching the paramters
Format: [{ title: string, movie run time: string, review rating: number, release date: string, movie rating: string }]

Example

Request: /getMoviesByRuntime?greaterOrLess=>&runtime=150
Result: [{"Title":"Avengers: Endgame","Movie Run Time":"181 min","Review Rating":8.8,"Release Date":"26-04-2019","Movie Rating":"PG-13"}]

/addMoviePOST

Body Parameters

titletype: string (required) - The name of the movie
phasetype: string (required) - The phase of the MCU the movie falls into
genrestype: string (required) - A comma seperated string of the movie's genres
release datetype: string (required) - The date the movie came out
director(s)type: string (required) - A comma seperated string of the movie's directors
screen writer(s)type: string (required) - A comma seperated string of the movie's screen writers
producer(s)type: string (required) - A comma seperated string of the movie's producers
casttype: string (required) - A comma seperated string of the movie's lead actors
plottype: string (required) - A short desciption of the movie
language(s)type: string (required) - A comma seperated string of languages spoken in the movie
filming locationtype: string (required) - The primary location where the movie was filmed
statustype: string (required) - The status of the movie's development
movie ratingtype: string (required) - The parental guidance rating of the movie
review ratingtype: number (required) - The rating the movie was given on IMDB (out of 10)
runtimetype: number (required) - The length of the film in minutes
budgettype: number (required) - The amount of money the Marvel Studios had to produce the movie
box officetype: number (required) - The gross worldwide amount of money the movie made

Returns

An MCU movie object representing the added movie
Format: { id: number, title: string, phase: string, genres: [string], release date: string, directors: [array], screenwriters: [string], producers: [string], cast: [string], plot: string, languages: [string], filming location: string, status: string, movie rating: string, review rating: number, movie run time: string, budget: number, box office: number }

/editStatusPOST

Query Parameters

idtype: number (required) - The ID of the movie you want to change
statustype: string (required) - The status that you are making the movie

Returns

Nothing gets returned (204 status codes are "no content" responses)

Example

Request: /editStatus
Body: { id: 24, status: 'Released' }
